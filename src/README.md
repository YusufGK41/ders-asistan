# 📚 Ders Asistanım - Akıllı Çalışma Planlayıcı

Ders Asistanım, öğrencilerin karmaşık çalışma süreçlerini düzenlemek, konuları takip etmek ve haftalık çalışma planlarını optimize etmek için geliştirilmiş modern bir web uygulamasıdır.

## 🚀 Özellikler (MVP-1)

- **Ders ve Konu Yönetimi:** İstenilen zorluk derecesinde ders ve alt konular ekleme.
- **İlerleme Takibi:** Biten konuları işaretleme ve görsel geribildirim.
- **Akıllı Kapasite Kontrolü:** Kullanıcının günlük/haftalık çalışma kapasitesine göre iş yükü hesaplaması.
- **Dinamik Planlama:** Açgözlü (Greedy) algoritma temelli taslak çalışma programı oluşturma (Genetik Algoritma entegrasyonu yolda!).
- **Modern Arayüz:** Tailwind CSS ile tasarlanmış, sekmeli ve kullanıcı dostu Dashboard (Panel) görünümü.
- **Yerel Depolama (Local Storage):** Verilerin tarayıcıda güvenle saklanması.

## 🛠️ Kullanılan Teknolojiler

- **Frontend:** React.js, Vite
- **Stil:** Tailwind CSS
- **State Yönetimi:** React Hooks (useState, useEffect)
